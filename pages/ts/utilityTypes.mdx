## 6.3.1. Partial

```ts showLineNumbers filename="partial.ts"
type UserType = {
  name: string
  age: number
  isMarried: boolean
}

type UtilityType = Partial<UserType>
```

## 6.3.2. Required

```ts showLineNumbers filename="required.ts"
type UserType = {
  name?: string
  age?: number
  isMarried?: boolean
}

type UtilityType = Required<UserType>
```

![](./images/required.webp)

## 6.3.3. Omit

```ts showLineNumbers filename="omit.ts"
type UserType = {
  name: string
  age: number
  isMarried: boolean
}

type UtilityType = Omit<UserType, 'name' | 'isMarried'>
```

![](./images/omit.webp)

## 6.3.4. Pick

```ts showLineNumbers filename="pick.ts"
type UserType = {
  name: string
  age: number
  isMarried: boolean
}

type UtilityType = Pick<UserType, 'name' | 'age'>
```
