## 6.3.2. Required

```ts showLineNumbers filename="required.ts"

  name?: string
  age?: number

```

## 6.3.3. Omit

```ts showLineNumbers filename="omit.ts"
type User = {
  name: string
  age: number
  isMarried: boolean
}

type Utility = Omit<User, 'name' | 'isMarried'>
```

![](./images/omit.webp)

## 6.3.4. Pick

```ts showLineNumbers filename="pick.ts"

  isMarried: boolean

type Utility = Pick<User, 'name' | 'age'>
```
